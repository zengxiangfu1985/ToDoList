ToDoList 绿色免安装版
=====================

【使用方法】
1. 将整个文件夹复制到任意位置（U 盘、桌面等均可）
2. 双击 ToDoList.exe 启动，或双击「启动 ToDoList.bat」
3. 无需安装 .NET / Qt / 其他运行库

【数据存储】
- 任务数据库、设置、日志均保存在本目录下的 data\ 文件夹
- settings.ini：应用设置（快捷键、语言、LLM 配置、M365 令牌等）
- tasks.db：任务数据库
- password.json：锁屏密码（若已启用）
- 拷贝整个文件夹即可带走全部数据

【系统要求】
- Windows 10 / 11，64 位
- 建议屏幕分辨率 1280×720 及以上

【说明】
- 首次运行会自动创建 data 目录
- 若从旧版升级，程序会自动将原注册表设置迁移到 data\settings.ini
- 若杀毒软件误报，请添加本文件夹为信任项
- 版本号见程序内「设置 → 关于 → 版本号」
